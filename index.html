🏗️ 건설현장 안전관리 - Electron 래퍼

현재 본인의 HTML이 아직 드롭인되지 않은 상태입니다.

1. 자동 가져오기 (권장)

npm run import -- "/경로/발표-품안공_추적관리_17.html"

→ CDN(Chart.js, SheetJS) 참조를 자동으로 로컬 vendor/ 경로로 치환하고 renderer/index.html에 복사합니다.

2. 수동 가져오기

  1. 본인 HTML을 renderer/index.html로 복사 (덮어쓰기)
  2. HTML 안의 다음 두 줄 수정: 
    <script src="https://cdnjs.cloudflare.com/.../chart.umd.min.js">
↓
<script src="./vendor/chart.umd.min.js">

<script src="https://cdnjs.cloudflare.com/.../xlsx.full.min.js">
↓
<script src="./vendor/xlsx.full.min.js">
✓ 벤더 라이브러리는 이미 renderer/vendor/에 번들되어 있으므로 오프라인/외장하드 환경에서 정상 동작합니다.
✓ 가져오기 후 앱을 재시작하면 본인 화면이 표시됩니다.
